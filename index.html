Current Version: 1.0

Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://Dirtywaterbrewing.github.io/repo/ (Give it the name "DWB Github").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "DWB Github", and install repository.Dirtywaterbrewing-1.0.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "DWB Repo"
  • Go into the "Program add-ons" section in the repo, and you'll find The DWB Wizard

Enjoy!
